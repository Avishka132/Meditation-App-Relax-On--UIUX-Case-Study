# 🌿 Relax ON – Inclusive UI/UX Design Case Study

> An inclusive UI/UX design project focused on creating an accessible, user-friendly, and engaging mental wellness application.

---

## 📖 Overview

**Relax ON** is a mental wellness and meditation mobile application designed using **Inclusive Design** and **User-Centered Design (UCD)** principles. The project aims to improve accessibility, usability, and overall user experience for people with diverse abilities and backgrounds.

The design emphasizes intuitive navigation, accessibility, and a calming user interface to help users maintain their mental well-being.

---

## 🎯 Project Objective

To design an intuitive, accessible, and inclusive mobile application that promotes mental wellness while ensuring a seamless experience for users with different accessibility needs.

## ♿ Accessibility Considerations

This project follows inclusive design practices by considering:

- High color contrast
- Readable typography
- Clear navigation
- Large touch targets
- Consistent layouts
- Simple interaction flow
- User-friendly interface for diverse users

---

## 📈 Project Outcome

This project demonstrates how **Inclusive Design Principles** can be applied to develop a mental wellness application that is accessible, user-friendly, and engaging.

It showcases the complete UI/UX design process—from research and personas to high-fidelity prototyping—while emphasizing accessibility and usability for a broad range of users.
